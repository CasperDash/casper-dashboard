## Getting Started

### Prerequisites

-   npm >= 7.24
-   nodejs >= 12

# Client

### Install

```sh
cd YOUR_WORKING_DIRECTORY/casper-dashboad/client
npm install
```

### Usage

Update API endpoint at `src/constants/key.js`

```shell
npm start
```

# API

### Install

```sh
cd YOUR_WORKING_DIRECTORY/casper-dashboad/server
npm install
```

### Usage

RPC url can be updated at `.env` or `constants/index.js`

```sh
npm start
```
